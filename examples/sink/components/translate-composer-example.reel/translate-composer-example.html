TranslateComposer

The TranslateComposer listens for touch/mouse events on an element or component and then calculates properties based on the mouse movement, such as momentum and bounce. You can then bind these calculated properties to another element or component, such as a slider control, for example.The following example uses the TranslateComposer to drag an element around the screen.

  • HTML
    <div class="wrap">
    <div class="mover" data-montage-id="example">Drag me</div>
</div>
<div class="controls">
    Axis:
    <select id="axis" required>
        <option value="both" selected="true">Both</option>
        <option value="horizontal" >Horizontal</option>
        <option value="vertical">Vertical</option>
    </select>
    <div>
        <label for="cb1"><input type="checkbox" data-montage-id="checkboxMomentum" id="cb1" /> Momentum </label>
    </div>
    <div>
        <label for="cb2"><input type="checkbox" data-montage-id="checkboxBouncing" id="cb2" /> Bouncing </label>
    </div>
    <code>translateX: <span id="x_value"></span></code>
    <code>translateY: <span id="y_value"></span></code>
</div>

        }
