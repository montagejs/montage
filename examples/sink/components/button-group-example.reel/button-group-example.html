Button Group

The ButtonGroup presents a styled list of Button components in a row. It supports icon-only mode for buttons with image icons.

HTML
<div id="buttonGroup1">
    <div id="button1_1" class="submit icon"><img src="images/play.png"/>Run</div>
    <div id="button1_2" class="text icon"><img src="images/upload.png" />Save</div>
    <div id="button1_3" class="text disabled">Delete</div>
    <div id="button1_4" class="cancel">Cancel</div>
</div>

CSS
The ButtonGroup supports a compact mode that you can enable by adding "small" to the ButtonGroup element's CSS class list. 
<div id="buttonGroup2" class="small">
    <div id="button2_1" class="submit icon"><img src="images/play.png"/>Run</div>
    <div id="button2_2" class="text icon"><img src="images/upload.png" />Save</div>
</div>
