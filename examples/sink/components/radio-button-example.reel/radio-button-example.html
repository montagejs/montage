RadioButton

The RadioButton component wraps an HTML radio utton input type with custom styles and functionality.

    <h5>Overall, how satisfied are you with our product?</h5>

    <tbody id="list2-rows">
        <tr>
             <td><input type="radio" id="answer-selection" name="answer-selection" /></td>

    </tbody>

    <h5>You selected: </h5>
    <div id="selected-answer" ></div>
</div>
