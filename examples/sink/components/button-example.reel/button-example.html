Button

The Button component converts a <div> or <input> element into a styled, clickable button. It supports three behavior modes (transient, toggle, and mixed) for different uses. You can different assign labels to different button states.

Handling Button Click/Touch Events

To setup a handler for a Button, implement a function with the following naming convention: 

"handle" + < button Identifier capitalized > + "Action"

Eg: for button with identifier = "button1": 

handleButton1Action: {
    value: function() {
        this.log("Button - button1 - clicked");
    }
}
