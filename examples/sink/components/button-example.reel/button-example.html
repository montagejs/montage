Button

The Button component converts a <div> or <input> element into a styled, clickable button. It supports three behavior modes (transient, toggle, and mixed) for different uses. You can different assign labels to different button states.

Controller code (partial)

The controller for this example defines an event handler for this button. The name of the handler that Montage invokes is derived from a concatenation of three strings:

"handle"
The phase that the event is being observed for on the button target.
"Button1"
The value of the identifier property assigned to the button (see serialization above), with its first letter capitalized.
"Action"
The event type that was registered.

var Montage = require("montage/core/core").Montage,
    Component = require("montage/ui/component").Component;

exports.ButtonExample = Montage.create(Component, {

    handleButton1Action: {
        value: function() {
            this.log("Button - button1 - clicked");
        }
    },

});

Handling Button Click/Touch Events

To setup a handler for a Button, implement a function with the following naming convention: 

"handle" + < button Identifier capitalized > + "Action"

Eg: for button with identifier = "button1": 

handleButton1Action: {
    value: function() {
        this.log("Button - button1 - clicked");
    }
}
