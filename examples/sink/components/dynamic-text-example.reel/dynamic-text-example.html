Dynamic Text

The DynamicText component lets you display a string of text in a DOM element.

Demo

This example uses a DynamicText component to display the current time, which is provided by the component's controller. Lastly, a DateConverter is used to format the time value in the hh:mm:ss format.

  • HTML markup 
    <span id="current-time" class="current-time"></span>
