Image

The Image component wraps an <img> element and exposes a src property to set the image's URL source.

Select an image from the list:

Serialization

Bind the main image's "src" property to the selected value of the ArrayController.

"main-image": {
    "module": "montage/ui/image.reel",
    "name": "Image",
    "properties": {
        "element": {"#": "img-main"}
    },
    "bindings": {
        "src": {
            "boundObject": {"@": "images"},
            "boundObjectPropertyPath": "selectedObjects.0.src",
            "oneway": true
        }
    }
}
