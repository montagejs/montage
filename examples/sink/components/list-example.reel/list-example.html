List

The List component is composed of a Repetition component inside of a Scroller component. Like the Repetition component, the List component iterates over a collection of data objects and generates new DOM structure for each item in its collection. You can configure a List to allow scrolling vertically, horizontally, or in both directions.

Vertical scrolling list

This example uses an ArrayController to provide the list its data.

  • HTML
    <div class="v-list" data-montage-id="v-list">
    <div class="listitem">
        <!-- Image and DynamicText components -->
        <img data-montage-id="thumb" class="thumb" />
        <span data-montage-id="title" class="title"></span>
    </div>
</div>

Horizontal scrolling list

This example provides one way to create a horizontal scrolling list using an unordered list.

  • HTML
    <ul data-montage-id="h-list" class="h-list">
  <li>
        <span id="title" class="title">
        <img class="thumb" data-montage-id="thumb-h"/>
  </li>
</ul>
